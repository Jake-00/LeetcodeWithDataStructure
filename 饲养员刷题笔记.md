485. ##### 最大连续1的个数

解法一：更新最大和

```
func(nums)->int:
    if nums is null or nums.length == 0
    	return 0
    count = 0
    result = 0
    for (i = 0; i < nums.length; i = i+1)
    	if nums[i] == 1
    		count = count+1
    	else
    		result = max(result,count)
    		count = 0
    result = max(result,count)
    return result		
```
